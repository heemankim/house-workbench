# Workbench Workspace: my-project

Sample workspace for Workbench plugin.

## 설정
- Vault: obsidian/
- Task Manager: local
- Plugin: house-workbench

## 명령어

| 명령어 | 설명 |
|--------|------|
| `/wb-start` | Task 시작 |
| `/wb-done` | 서브태스크 완료 |
| `/wb-next` | 다음 서브태스크 |
| `/wb-end` | Task 종료 |
| `/wb-status` | 상태 조회 |
| `/wb` | 도움말 |
