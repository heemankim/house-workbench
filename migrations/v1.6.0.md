# v1.6.0: FTS5 Vault 인덱서 (Beta)

## 변경 요약

| 항목 | 이전 | 이후 |
|------|------|------|
| Vault 검색 | 2-stage grep (Stage 1: tags, Stage 2: TL;DR) | FTS5 또는 grep (beta 플래그) |
| config.json | `beta` 블록 없음 | `beta: {}` 블록 추가 |
| 토큰 소모 | 검색 시 파일 전체 읽기 | FTS5: 경로+TL;DR만 반환 (~88% 절감) |
| 검색 도구 | Grep + Read | Bash 1회 (python3 vault-search.py) |

### 배경

Vault 검색 시 grep 매칭 → 파일 전체 읽기로 토큰 대량 소모.
SQLite FTS5로 인덱싱하면 검색 전 파일 내용을 읽지 않고도 관련 문서를 찾을 수 있음.

## 자동 마이그레이션

### 1. config.json에 beta 블록 추가

config.json에 `beta` 키가 없으면 추가:

```json
{
  "beta": {}
}
```

### 2. tools/vault-search.py 배치

플러그인 캐시의 `tools/vault-search.py`를 workspace `tools/`에 복사:

```bash
mkdir -p tools
cp {plugin-cache}/tools/vault-search.py tools/vault-search.py
```

### 3. .gitignore 업데이트

`.gitignore`에 DB 파일 추가:

```
tools/vault-index.db
```

## 수동 확인

FTS5 인덱서를 활성화하려면 config.json에서 설정:

```json
{
  "beta": {
    "fts-indexer": true
  }
}
```

활성화 후 초기 인덱싱:

```bash
python3 tools/vault-search.py index --vault {vault-path}
```
