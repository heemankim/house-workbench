# v1.8.0: 한국어 토크나이저 (kiwipiepy)

## 변경 요약

| 항목 | 이전 | 이후 |
|------|------|------|
| FTS5 토크나이저 | unicode61 (공백 기반) | kiwipiepy pre-tokenize + unicode61 |
| 한국어 복합어 | 검색 불가 ("자산관리" → 0건) | 형태소 분리 후 매칭 ("자산 관리") |
| DB schema | v1 | v2 (자동 재구축) |
| 외부 의존성 | 없음 | kiwipiepy (선택, fallback 지원) |

### 배경

FTS5 기본 unicode61 토크나이저는 공백 기반으로 한국어 복합어를 분리하지 못함.
kiwipiepy 형태소 분석기로 인덱싱 시 pre-tokenize하여 복합어 검색을 지원.

## 자동 마이그레이션

### 1. tools/vault-search.py 업데이트

플러그인 캐시의 최신 vault-search.py를 workspace tools/에 복사:

```bash
cp {plugin-cache}/tools/vault-search.py tools/vault-search.py
```

### 2. DB 자동 재구축

schema_version이 v1이면 다음 검색/인덱싱 시 자동으로 DB를 재구축합니다.
수동 재인덱싱도 가능:

```bash
python3 tools/vault-search.py index --vault {vault-path}
```

### 3. kiwipiepy 설치 (선택)

한국어 토크나이저를 활성화하려면:

```bash
pip install kiwipiepy
```

미설치 시 기존 unicode61 토크나이저로 동작합니다 (v1.6.0과 동일).

## 수동 확인

`python3 tools/vault-search.py status`로 확인:

```json
{
  "schema_version": "2",
  "korean_tokenizer": true
}
```

- `schema_version: "2"` — pre-tokenize 적용된 인덱스
- `korean_tokenizer: true` — kiwipiepy 활성화됨

## What's New

```
한국어 토크나이저 (kiwipiepy pre-tokenize)
├─ 한국어 복합어 검색 지원 ("자산관리", "피부관리", "운동루틴")
├─ kiwipiepy 형태소 분석으로 인덱싱 시 자동 분리
├─ DB schema v2 (자동 마이그레이션, 재인덱싱)
├─ 외래어 과분리 방지 (연속 단일 글자 재결합)
├─ kiwipiepy 미설치 시 기존 동작 유지 (graceful fallback)
└─ vault-conventions 스킬에 한국어 토크나이저 문서 추가
```
